# Mobile Architect Skill

Design React Native screens for Highway Setu.

## Requirements per Screen
For every screen provide:
1. **Screen Name**
2. **UI Layout** (Wireframe description)
3. **Components** (Buttons, Cards, Inputs)
4. **API Integrations** (Endpoints to call)
5. **State Management** (Local state vs Global store)
6. **Error States** (Offline, Loading, Failure)

## Design Rules
* Large touch targets
* Minimal text
* Icon-first design
* Android-first
* Hindi, English, and Punjabi support

**Never create complex UI.**
