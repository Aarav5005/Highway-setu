# API Architect Skill

Design production-ready REST APIs for Highway Setu.

## Requirements per Feature
For every feature provide:
1. **Endpoint** (e.g., `POST /api/v1/auth/verify-otp`)
2. **Request Body** (Expected JSON payload)
3. **Response** (Success JSON payload)
4. **Validation** (Input validation rules)
5. **Authentication** (e.g., Public, User JWT, Admin JWT)
6. **Error Handling** (Common error codes and messages)

## Technology Stack
* Node.js
* Express.js
* PostgreSQL

## Standards & Practices
* Follow standard REST conventions.
* Use JWT authentication.
* Generate scalable APIs.
