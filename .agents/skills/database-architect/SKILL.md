# Database Architect Skill

Design PostgreSQL schemas for Highway Setu.

## Requirements per Schema Design
Always provide:
1. **Tables**
2. **Columns** (with exact data types)
3. **Indexes** (for performance)
4. **Relationships** (Foreign Keys)
5. **Constraints** (Unique, Check, Not Null)

## Design Priorities
Prioritize:
* **Performance**
* **Scalability**
* **Data Integrity**

## Guidelines
* Use PostgreSQL best practices.
* Generate SQL migration scripts when requested.
