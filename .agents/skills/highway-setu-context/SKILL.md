# Highway Setu Project Context

Highway Setu is a mobile-first platform connecting Truck Drivers, Dhaba Owners, and Mechanics across Indian highways.

## Primary Goal:
Help truck drivers discover trusted dhabas, mechanics, and essential services while traveling.

## Target Users:
* Truck Drivers
* Dhaba Owners
* Mechanics
* Admin Team

## Supported Languages:
* English
* Hindi
* Punjabi

## Design Principles:
* Low literacy friendly
* Large buttons
* Icon-first UI
* Minimal typing
* Voice input where possible
* Works on weak internet connections
* Android-first experience

## Version 1 Features:

### Driver:
* OTP Login
* Profile Management
* Truck Details
* Start Journey
* Live GPS Tracking
* Vendor Discovery
* Distance Calculation
* Vendor Details
* Call Vendor
* Open Google Maps Navigation

### Dhaba:
* Registration
* FSSAI Verification
* Menu Management
* Amenities Management
* Order Management

### Mechanic:
* Registration
* Service Listing
* Availability Management
* Job Requests

### Admin:
* User Verification
* Vendor Approval
* Analytics Dashboard
* User Management

## Tech Stack:
* **Mobile:** React Native
* **Backend:** Node.js, Express.js
* **Database:** PostgreSQL
* **Maps:** Google Maps
* **Authentication:** Firebase OTP
* **Notifications:** Firebase Cloud Messaging
* **Admin Dashboard:** Next.js, Tailwind CSS

## Core Priorities:
Always prioritize:
1. Simplicity
2. Reliability
3. Scalability
4. Mobile Performance
5. Ease of Use

## Design Guidelines:
When designing any feature:
* Define user flow
* Define screens
* Define APIs
* Define database changes
* Define edge cases

**Never over-engineer.**
**Build production-ready solutions.**
